A userscript to insert a newline in the Niconico Douga comment field with Shift + Enter.

## Usage

### Installation

1. Install any userscript manager
2. Click `nico-comment-line-break.user.js` in the `Assets` section of the [latest release](https://github.com/KOTO28/nico-comment-line-break/releases/latest)
3. Follow the instructions from your userscript manager to install

### How to Use

1. Open any video on Niconico Douga
2. While typing a comment, press `Shift` and `Enter`
3. A line break is inserted into the comment you are typing.
