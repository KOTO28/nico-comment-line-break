ニコニコ動画のコメント入力欄で Shift + Enter で改行できるようにするユーザースクリプト

## 使い方

### インストール

1. 任意のユーザースクリプトマネージャーをインストール
2. [最新のリリース](https://github.com/KOTO28/nico-comment-line-break/releases/latest)の `Assets` にある `nico-comment-line-break.user.js` をクリック
3. ユーザースクリプトマネージャーの指示に従ってインストール

### 使用方法

1. ニコニコ動画の任意の動画を開く
2. コメント入力中に `Shift` キーを押しながら `Enter` キーを押す
3. 入力中のコメントに改行が挿入されます。

## 注意事項

<!-- - Firefox + Violentmonkey および Google Chrome + Tampermonkey での動作を確認しています。 -->
- ニコニコ動画のページ構造が変更された場合、スクリプトが正常に動作しなくなる可能性があります。
